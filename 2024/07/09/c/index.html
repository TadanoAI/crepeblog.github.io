C++工作原理：

本章节中将要介绍C++代码是如何工作，从源文件中开始学习编写到实际的可执行的二进制代码,

编写C++程序基本流程

将C++源文件给编译成二进制的代码。可能是编译某种库或者可执行程序

本章节主要讲可执行程序

1.创建hello world源码

在vs中创建一个空白的项目Hello World

在资源文件中创建main.cpp

右键Hello World项目生成U将会在路径.\HelloWorld\x64\Debug中生成一个名为Hello World.exe文件，双击执行

2.源码解释

在这段上述这段中，部分指令注释：

指令 注释
#include 包含头文件，用于引用库函数或者头文件内容
int main() 主函数入口，程序从这里开始执行
std::cout 标准输出流，用于输出数据到控制台
std::endl 用于在输出流中插入一个换行符，并刷新输出缓冲区。等效于‘\n’加上std::flush
std::cout.get() 从标准输入流（控制台）中读取一个字符，并返回其ASCII值
std::cout 标准输出流对象，用于将数据输出到控制台

这是一段非常基础的程序但是包含了很多知识点

#include <iostream

首先#include <iostream>叫做预处理，在#符号号之后的都是预处理语句，编译器收到源文件后，会先阅读这一条语句，优先处理预处理语句【他在编译发生之前就先被处理了所以叫预处理】

iostream

include的含义就是把它包含的文件复制黏贴过来到当前命令所在行，在该例子中需要找到名为iostream的文件，把它包含的文件所有内容拷贝到这个命令行内，类似于iostream这些类型的文件被称为头文件

之所以要调用iostream这个头文件是因为我们需要用到一个被调用的函数的声明std::cout，让我们可以在终端中打印东西

main函数

在任何C++程序中都必须包括main函数，main函数是整个程序的入口，意思是当我们运行程序时，计算机从这个函数开始执行代码，当程序开始运行时，计算机会逐条执行我们的代码，当然，程序也可以中断或者改变执行的顺序，一般通过控制语句或者函数调用来改变。但是最主要还是一行一行执行

至此，我们的程序首先执行的是std::cout << "Hello world!" << std::endl;这段。然后执行std::cin.get();我们的程序就执行结束了。

int整数类型

如果写过别的代码会发现，main的函数返回值是int【整数类型】，然而我们并没有返回int，因为main函数比较特殊，它不一定要返回值，如果不返回值，则默认返回了0，这个只在main函数中适用。

<<重载运算符

程序中有<<符号，只是写成这样子，实际并没有特别含义，这些看上去像左移运算符的东西成为重载运算符，可以把它理解成一个函数，但并不是运算符,差不多是std::cout.print("Hello world").print(std::endl);

将”Hello world“字符串推送到cout流中，然后打印到终端，后面是行结束符号，跳转下一行

std::cin.get()指令

这段代码中std::cin.get();是在前往下一句代码之前等待输入一次enter键，程序暂停执行，直到按下回车后程序才继续运行到下一行，由于没有下一行了，所以程序返回0，就意味着代码执行完了
